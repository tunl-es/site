Expose HTTP services to the internet

NodeJS service tunning at your localhost on port 3000? Or an Python Django project at port 8000? Use the tunl http to create a public and secure internet address for it or any other http service in the local network.

$ tunl http localhost:3000
https://red-fox.tunl.es -> http://localhost:3000

$ tunl tcp localhost:3306
tcp.tunl.es:23011 -> localhost:3306

Expose TCP services to the internet

MySQL instance running on your local machine at port 3306? Or an MongoDB at 27017? Share this or any other TCP service with the tunl tcp command to create a fast and public internet addres for it.
